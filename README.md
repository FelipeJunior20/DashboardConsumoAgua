# DashboardConsumoAgua

Este projeto apresenta um fluxo completo de análise de dados aplicado ao consumo de água. Os dados são gerados e tratados por scripts Python, organizados em arquivos CSV, e posteriormente visualizados em um dashboard interativo no Power BI.
O processo segue uma abordagem de ETL:

Geração dos dados com gerarDados.py, criando um conjunto de consumo fictício.

Tratamento dos dados com tratarDados.py, realizando limpeza, padronização e formatação.

Visualização dos dados no Power BI através do arquivo dashboardAgua.pbix, com gráficos, KPIs e análises comparativas.

O projeto demonstra um ciclo completo de manipulação de dados da criação à visualização utilizando Python para processamento e Power BI para análise visual.
